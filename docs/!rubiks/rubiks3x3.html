À propos

Le Rubik's cube a été inventé par Ernő Rubik, d'où il tire son nom.
Il s'agit du jouet de plus vendu de tous les temps, avec 350 million d'unités vendues depuis 1980.

La méthode de résolution pour débutant a 7 étapes.

1. Résoudre la croix blanche

But: Placer tous les bords (pièces avec 2 couleurs) à côté du centre blanc.
Veiller à ce que les couleurs de l'autre côté des bords soient également alignés avec les autres centres.

  • Trouver la pièce centrale blanche et la placer sur le dessus.
  • Chercher un bord blanc — une pièce qui a deux couleurs, le blanc et une autre couleur.
  • Si cette pièce est sur la couche inférieure
    • Tourner la face inférieure jusqu'à ce que la pièce se trouve sous le centre de la même couleur.
      Si le bord est blanc et rouge, alors la placer sous le centre rouge.
    • Tourner cette face jusqu'à ce que le bord blanc fasse partie de la croix blanche.
  • Si la pièce est sur la couche supérieure (c'est la bonne position mais les couleurs sont inversées).
    • La tenir en face
    • Tourner la face frontale vers la droite, pour placer le bord blanc sur le côté droit
    • Tourner la face supérieure dans le sens inverse des aiguilles d'une montre — vers la droite
    • Tourner le bord droit vers le haut
    • Tout remettre en place: tourner la face supérieure dans le sens des aiguilles d'une montre — vers la gauche
  • Si la pièce se trouve sur la couche intermédiaire: la déplacer vers la couche inférieure
    • Tourner le bord blanc vers la face inférieure
    • Tourner la couche inférieure pour écarter le bord blanc
    • Remettre la face à sa position initiale
    • Maintenant que le bord blanc est sur la couche inférieure, le résoudre comme d'habitude
  • Répéter jusqu'à ce que le croix blanche soit entièrement résolue, les couleurs sur les côtés doivent également être alignées avec les centres de même couleur.

2. Résoudre la couche blanche

Maintenant que les bords sont résolus, résoudre les coins (pièces avec 3 couleurs).

  • Retourner le cube: centre jaune vers le haut.
  • Chercher un coin blanc dans la couche supérieure
    • Regarder les autres couleurs: si la pièce est vert/blanc/orange, elle va entre les centres vert/blanc/orange
    • Tourner la couche supérieure jusqu'à ce que le coin soit juste au-dessus de l'emplacement où il doit aller
    • Si le blanc est sur le côté droit: utiliser l'algorithme de la main droite en 4 mouvements
      • Face droite vers le haut
      • Face supérieure vers la gauche
      • Face droite vers le bas
      • Face supérieure vers la droite
    • Si le blanc est sur le côté gauche: main gauche en 4 mouvements
      • Face gauche vers le haut
      • Face supérieure vers la droite
      • Fauche gauche vers le bas
      • Face supérieure vers la gauche
    • Si le blanc est sur la face supérieure:
      répéter le même algorithme (gauche ou droite) jusqu'à ce que le coin soit résolu — prendra jusqu'à 3 répétitions
  • Si le coin blanc est sur la couche inférieure mais que les couleurs ne s'alignent pas:
    mettre n'importe quel coin à sa place, puis résoudre le coin blanc comme d'habitude.
  • Répéter jusqu'à avoir résolu la couche blanche.
    Vérifier que toutes les couleurs sur les côtés sont correctes, puis passer à l'étape suivante.

3. Résoudre la 2ème couche

  • La couche supérieure est jaune, donc trouver un bord sur la couche supérieure qui n'a pas de jaune
    • Déplacer la pièce pour que la couleur sur le côté soit de la même couleur que le centre:
      Si la pièce est bleue sur la face supérieure et orange sur le côté, tourner jusqu'à ce que le orange s'aligne avec le centre orange
    • Déterminer dans quelle direction la pièce doit aller:
      si la pièce en cours est alignée avec le centre orange, et que 2ème couleur est le bleu, elle doit aller du côté du centre bleu — ici à droite
      • Pour l'insérer à droite
        • Réserver la pièce sur le côté gauche: tourner la couche supérieure vers la gauche
        • Sortir le coin blanc avec la main droite: côté droit vers le haut, tourner le haut vers la gauche, côté dront vers le bas, tourner le haut vers la droite
        • Réinsérer le coin avec la main gauche: côté gauche vers le haut, tourner le haut vers la droite, côté gauche vers le bas, tourner le haut vers la gauche
      • Pour l'insérer à gauche
        • Réserver la pièce sur le côté droit: tourner la couche supérieure vers la droite
        • Sortir le coin blanc avec la main gauche: côté gauche vers le haut, tourner le haut vers la droite, côté gauche vers le bas, tourner le haut vers la gauche
        • Réinsérer le coin blanc avec la main droite: côté droit vers le haut, tourner le haut vers la gauche, côté droit vers le bas, tourner le haut vers la droite
  • Si la pièce est déjà sur la 2ème couche, mais que les couleurs ne s'alignent pas:
    mettre n'importe quel bord supérieur à sa place et continuer comme d'habitude.

4. Créer la croix jaune

Une fois les deux premières couches résolues, créer la croix jaune. Ignorer les coins et ne pas chercher à résoudre les bords: les couleurs sur les côtés n'ont pas d'importance, seul le jaune sur la face supérieure importe.

  • Cas n°1: aucun bord
    • Tourner la face frontale vers la droite
    • Algorithme de la main droite: face droite vers le haut, face supérieure vers la gauche, face droite vers le bas, face supérieure vers la droite
    • Tourner la face frontale vers la gauche
  • Cas n°2: L
    • En cas de forme en L, le placer en haut à gauche (forme un ⮥)
    • Faire la même chose que le cas n°1
  • Cas n°3: ligne
    • En cas de ligne, la mettre à l'horizontal (forme un ―)
    • Faire la même chose que le cas n°1

5. Résoudre la croix jaune

Maintenant, il faut que toutes les couleurs de la croix jaune soient correctes sur les côtés.

  • Essayer d'aligner autant de pièces que possible.
    Ce sera soit 4 soit 2.
  • Si possible, tenir les deux bords résolus sur le côté droit et à l'arrière.
    S'ils sont en face l'un de l'autre, faite l'algorithme à partir de n'importe où.
    • Couche droite vers le haut
    • Couche supérieure vers la gauche
    • Couche droite vers le bas
    • Couche supérieure vers la gauche
    • Couche droite vers le haut
    • Couche supérieure vers la droite
    • Couche supérieure vers la droite
    • Couche droite vers le bas
    Si c'est fait correctement, il devrait être possible de tout aligner.

6. Placer les coins jaunes

Placer tous les coins jaunes au bon endroit.

  • Chercher les coins qui sont au bon endroit (même si les couleurs ne s'alignent pas):
    le coin vert/orange/jaune va entre le vert/orange/jaune.
    Il y aura 4, 1 ou 0 coins corrects.
  • Si 1 coin est correct, le tenir sur l'avant-droit.
    Si aucun n'est correct, exécuter l'algorithme à partir de n'importe où.
    • Face supérieure vers la gauche
    • Face droite vers le haut
    • Face supérieure vers la droite
    • Face gauche vers le haut
    • Face supérieure vers la gauche
    • Face droite vers le bas
    • Face supérieure vers la droite
    • Face gauche vers le bas
  • Répéter si nécessaire

7. Résoudre les coins jaunes

Maintenant tous les coins jaunes sont à la bonne position, mais pas forcemment alignés correctement.

  • Retourner le cube: le centre blanc vers le dessus
  • Mettre le coin non résolu en bas à droite
    • Exécuter l'algorithme de la main droite: couche droite vers le haut, couche supérieure vers la gauche, couche droite vers le bas, couche supérieure vers la droite
    • Répéter ces 4 mouvements jusqu'à ce que le coin soit résolu.
      Ne pas oublier le dernier mouvement (couche supérieure vers la droite), même si le coin inférieur est résolu, autrement la couche supérieure sera mélangée à la fin.
  • Faites passer le prochain coin non résolu en bas à droite.
    Ne pas retourner le cube, simplement tourner la couche inférieure.
    Répéter l'étape précédente.
  • Résoudre tous les coins un par un jusqu'à ce qu'ils soient tous résolus
  • Tout aligner.
