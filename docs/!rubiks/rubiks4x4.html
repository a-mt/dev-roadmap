1. Résoudre le centre blanc

  • Trouver deux pièces centrales blanches et les joindre.
    Cela forme une barre
  • Trouver les deux autres pièces centrales blanches et les joindre aussi, sans casser la prmeière barre
  • Joindre les deux barres ensemble

2. Résoudre le centre jaune

  • Créer une barre jaune
    Si les deux pièces centrales jaunes sont en diagonales, les séparer et utiliser une rotation de manière à ce que, lorsqu'on les rejoint de nouveau, cela forme une autre barre.
  • Placer la barre jaune à l'opposé du centre blanc
  • Créer une deuxième barre jaune et la placer à l'opposé du centre blanc

3. Résoudre 3 bords blancs

Un bord est une pièce avec 2 couleurs — ce n'est ni une pièce centrale ni un coin

Premier bord

  • Trouver deux bords blancs, qui ont les même couleurs, comme blanc/bleu avec blanc/bleu.
  • Placer ces deux bords sur les 2 tranches du milieu, un sur la tranche du haut, un sur la tranche du bas.
  • Tourner une tranche pour rejoindre les deux bords blancs.
  • Effectuer une rotation pour mettre le bord blanc dans le côté blanc, blanc contre blanc.

Deuxième bord

  • Trouver une autre paire
  • Comme précédemment, placer ces deux bords sur les tranches du milieu avant de les rejoindre.
    Prêter attention au bord blanc résolu: les rotations ne doivent être effectuées que sur les faces où les bords ne sont pas déjà résolus (sinon, tourner la face supérieure pour les mettre à l'écart).
  • Mettre le nouveau bord blanc dans le côté blanc, blanc contre blanc.
    Respecter l'ordre des couleurs: BORG (Blue Orange Red Green) — le côté bleu à l'opposé du vert, orange à l'opposé du rouge.

Troisème bord

  • Même principe pour le troisième bord blanc, toujours en respectant l'ordre BORG.
  • On s'occupera du dernier bord blanc après avoir résolu les centres.

4. Résoudre les centres

  • Résoudre les 4 centres restants, en les alignant avec les côtés de la face blanche.
    Pour ne pas casser les 3 bords blancs résolus, les rotations doivent toujours être effectuées sur le côté qui n'a pas le bord blanc.
    Également prêter attention à n'effectuer les rotations qu'entre deux faces aux centres non résolus

5. Créer les bords

Appairer tous les bords, sans chercher à les placer au bon endroit.

Placer les bords au milieu

  • Commencer par le bord blanc restant
    Si le bord blanc se situe sur la face blanche, le déplacer vers la face jaune
  • Tenir la face jaune vers le haut.
  • Si un ou deux bords se situent sur la face supérieure:
    Placer les deux bords sur les tranches du milieu.
    Effectuer une rotation verticale, remplacer le bord qui s'y trouvait par le bord blanc, redescendre la face.

Appairer deux pièces du milieu

  • Regarder dans quelle tranche (ligne horizontale) se trouve les deux pièces.
  • Cas n°1: différentes tranches:
    • Les appairer en tournant le milieu. Cela casse les centres, mais on va les restaurer juste après.
    • Effectuer une rotation de la face où se situe le bord, pour le mettre dans la couche supérieure
    • Le remplacer par n'importe quel bord non résolu
    • Remettre la face à sa position précédente
    • Restaurer les centres
  • Cas n°2: même tranche:
    Que ce soit sur la tranche supérieure ou inférieure, l'algorithme est le même:
    • Tourner le milieu pour appairer la pièce du haut du premier bord, avec la pièce du bas du deuxième bord
    • Inverser le bord droit: R U R' F R' F' R
      • Couche droite vers le haut
      • Couche supérieure vers la gauche
      • Couche droite vers le bas
      • Couche frontale vers la droite
      • Couche droite vers le bas
      • Couche frontale vers le haut
      • Couche droite vers le haut
    • Restaurer les centres

Appairer les deux derniers bords non résolus

  • Cas n°1: même tranche: comme d'habitude
  • Cas n°2: différentes tranches:
    • Inverser le bord droit: R U R' F R' F' R
    • Résoudre les bords comme d'habitude — les couleurs sont sur la même ligne

6. Résoudre les coins blancs

Même principe qu'un Rubik's cube 3x3

7. Résoudre les couches intermédiares

Même principe qu'un Rubik's cube 3x3

8. Créer la croix jaune

  • Toujours le même principe qu'un Rubik's cube 3x3.
  • Si un bord est inversé: Parité OLL
    Dans certains cas, on peut finir avec un bord inversé — la couleur gauche est sur le côté et non sur la face supérieure. Si ça arrive, on utilise l'algorithme Parité OLL: cet algorithme retourne le bord en haut sur la face frontale, et inverse les deux bords tranverses sur la face supérieure.
    • Mettre le bord à inverser en face
    • Milieu droit vers le haut (r)
    • Couche supérieure vers la droite, deux fois (U2)
    • Retourner le cube pour avoir en haut la face frontale (x)
    • Milieu droit vers le haut (r)
    • Couche supérieure vers la droite, deux fois (U2)
    • Milieu droit vers le haut (r)
    • Couche supérieure vers la droite, deux fois (U2)
    • Milieu droit vers le bas (r')
    • Couche supérieure vers la droite, deux fois (U2)
    • Milieu gauche vers le bas (l)
    • Couche supérieure vers la droite, deux fois (U2)
    • Milieu droit vers le bas (r')
    • Couche supérieure vers la droite, deux fois (U2)
    • Milieu droit vers le haut (r)
    • Couche supérieure vers la droite, deux fois (U2)
    • Milieu droit vers le bas (r')
    • Couche supérieure vers la droite, deux fois (U2)
    • Milieu droit vers le bas (r')
  • S'assurer que les bords respectent l'ordre: bleu opposé à rouge, orange opposé à vert
  • Pour inverser deux bords l'un face à l'autre:
    Placer les deux bords à l'avant/arrière et appliquer l'algorithme de Parité PLL (Permutation of the Last Layer)
    • 2ème couche de droite vers le haut, deux fois (2R2)
    • Couche supérieure vers la droite, deux fois (U2)
    • 2ème couche de droite vers le haut, deux fois (2R2)
    • Moitié supérieure vers la droite, deux fois (u2)
    • 2ème couche de droite vers le haut, deux fois (2R2)
    • Moitié supérieure vers la droite, deux fois (u2)
    • Réaligner les coins
  • Pour inverser deux bords adjacents:
    Placer les deux bords à l'avant/côté droit et appliquer l'algorithme de Parité OLL (Orientation of the Last Layer)
    • Côté droit vers le haut (R)
    • Côté supérieur vers la gauche (U)
    • Côté droit vers le bas (R')
    • Côté supérieur vers la droite (U')
    • Parité PLL
    • Côté supérieur vers la droite (U')
    • Côté droit vers le haut (R)
    • Côté supérieur vers la droite (U')
    • Côté droit vers le bas (R')

9. Placer les coins jaunes

  • Même principe qu'un 3x3: mettre les coins jaunes au bon endroit, sans s'occuper de les aligner correctement avec les centres. Il est possible de se retrouver avec 2 coins résolus et 2 coins non résolus (ce qui est impossible avec un 3x3) — il s'agit d'un problème de parité.
  • Cas n°1: 2 coins non résolus en diagonale
    • Réarranger les coins (avec la technique utilisée dans le 3x3) pour avoir les 2 coins non résolus du même côté.
      Ensuite, appliquer le cas n°2 — ci-dessous
  • Cas n°2: 2 coins non résolus du même côté
    • Les mettre en face
    • Appliquer l'algorithme PLL (voir étape 10) — il inverse le bord à l'avant avec le bord à l'arrière.
    • Normalement, il est maintenant possible de placer les coins normalement. S'il reste deux coins non résolus, répéter l'algorithme
