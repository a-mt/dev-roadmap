## React côté serveur

React est une librairie JavaScript, on peut donc s'en servir côté serveur, avec Node. On utilise alors [`ReactDOMServer`](https://fr.reactjs.org/docs/react-dom-server.html).

``` js
ReactDOMServer.renderToString(<App />);
```
